# 云服务器网卡解绑虚拟IP地址<a name="ecs_02_0504"></a>

## 功能介绍<a name="section10723444"></a>

虚拟IP地址用于为网卡提供第二个IP地址，同时支持与多个弹性云服务器的网卡绑定，从而实现多个弹性云服务器之间的高可用性。

该接口用于解绑定弹性云服务器网卡的虚拟IP地址。解绑后，网卡不会被删除，如需删除云服务器网卡，请参见[批量删除云服务器网卡](批量删除云服务器网卡.md)。
