# 删除云服务器标签<a name="ZH-CN_TOPIC_0065820824"></a>

删除云服务器所有tag。

需在客户端通过以下HTTP header指定微版本号：X-OpenStack-Nova-API-Version: 2.26。

## URI<a name="zh-cn_topic_0057972839_section28406254"></a>

DELETE /v2.1/\{project\_id\}/servers/\{server\_id\}/tags
