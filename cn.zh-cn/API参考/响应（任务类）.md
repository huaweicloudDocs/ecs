-   响应示例

    正常响应：

    ```
    { 
        "job_id": "70a599e0-31e7-49b7-b260-868f441e862b"
    } 
    ```

    异常响应：

    ```
    { 
        "error": {"message": "", "code": XXX，""}
    } 
    ```

    包含错误详情的异常响应：

    ```
    {
        "error": {
            "message": "xxxx",
            "code": "xxxx",
            "details": [
                {
                    "code": "xxxx",
                    "message": "xxxx"
                }
            ]
        }
    }
    ```
