-   响应示例

    正常响应：

    ```
    { 
        "response": [
                      {
                        "id": "616fb98f-46ca-475e-917e-2563e5a8cd19"   
                      },
                      {
                        "id": "516fb98f-46ca-475e-917e-2563e5a8cd12"   
                      }
                   ]
    } 
    ```

    异常响应：

    ```
    {
         "error": {
                     "code": "Ecs.xxxx",
                     "message": "xxxxxxxxxxxxxxx" 
                   },
         "internalError": [
                     {
                        "id": "616fb98f-46ca-475e-917e-2563e5a8cd19",
                        "error_code": "ECS.XXXX",
                        "error_message": "xxxxxxxxxxxxxxx" 
                      },
                     {
                         "id": "516fb98f-46ca-475e-917e-2563e5a8cd12",
                         "error_code": "ECS.XXXX",
                         "error_message": "xxxxxxxxxxxxxxx" 
                     }
                  ]
    }
    ```
