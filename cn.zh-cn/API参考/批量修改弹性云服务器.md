# 批量修改弹性云服务器<a name="ecs_02_0305"></a>

## 功能介绍<a name="section941732182911"></a>

批量修改弹性云服务器信息。

当前仅支持批量修改云服务器名称，一次最多可以修改1000台。

## URI<a name="section85409429323"></a>

PUT /v1/\{project\_id\}/cloudservers/server-name
