# 查询弹性云服务器磁盘信息（废弃）<a name="ZH-CN_TOPIC_0101860613"></a>

## 功能介绍<a name="section61843920"></a>

当前API已废弃，请使用[查询弹性云服务器挂载磁盘列表信息](查询弹性云服务器挂载磁盘列表信息.md)。
