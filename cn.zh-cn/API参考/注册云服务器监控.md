# 注册云服务器监控<a name="ecs_02_1401"></a>

## 功能介绍<a name="section3492751018840"></a>

将云服务器添加到监控表中。

注册到监控表中的云服务会被ceilometer周期性采集监控数据，包括平台的版本、cpu信息、内存、网卡、磁盘、硬件平台等信息，这些数据上报给云监控。例如SAP云服务器内部的插件会周期性从云监控中查询监控数据，以报表形式呈现给SAP。
