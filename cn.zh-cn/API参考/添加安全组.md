# 添加安全组<a name="ZH-CN_TOPIC_0067161469"></a>

## 功能介绍<a name="zh-cn_topic_0057973179_section16588975"></a>

为弹性云服务器添加一个安全组。

添加多个安全组时，建议最多为弹性云服务器添加5个安全组。

## URI<a name="zh-cn_topic_0057973179_section15083054"></a>
